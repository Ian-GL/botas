(inner) formatIfExist(fn, placeholder, value) → {function|function|fn(value)|placeholder}

formatIfExist Composed function that returns

  • A placeholder if the value is not truthy or
  • A formatted value

The way to use this function is: i.e.

  • format the content of a variable. If the variable has a truthy content, run a formatter on it, if not, return '-'

    var amount = api.getAmount(); //=> 0.6778 formatIfExist(toPercentage)('-')(amount); //=> "67.78 %"

    var amount = api.getAmount(); //=> undefined formatIfExist(toPercentage)('-')(amount); //=> "-"
