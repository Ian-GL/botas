signGoogleMapsURL(url, secret) → {string}

Sign GoogleMapsStaticAPI url Signature generating algorithm: (extracted from https://developers.google.com/maps/documentation/maps-static/get-api-key#client-id)

  1. Generate query with clientID (signature is not added yet)
  2. Strip domain from request ( localhost:3000/ -- /maps/api/static?center=... )
  3. Retrieve private key, originally in Base64 Sign the URL using HMAC-SHA1. First decode the private key. Note that Base64 for URLS replaces + and / with - and _
  4. Encode the resulting binary signature using the modified Base64
  5. Attach the signature to the URL
